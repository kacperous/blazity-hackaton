# Project Vocabulary

Use this file to define canonical product and codebase terms for AI agents.

## Terms

| Term | Meaning | Avoid |
| --- | --- | --- |
| Atlas | The `@blazity-atlas/core` AI workspace tooling under `.ai/` | "the AI config" |
| Claude | The Anthropic model family powering the app's AI features | "the LLM", "GPT" |
| Anthropic SDK | `@anthropic-ai/sdk`, the client used to call Claude | "the AI library" |
| Proxy layer | Server-side code that holds the API key and calls Claude | "the backend" (when ambiguous) |
